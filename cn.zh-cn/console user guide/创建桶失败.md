# 创建桶失败<a name="zh-cn_topic_0068009037"></a>

## 问题<a name="section60232767"></a>

用户通过OBS创建桶失败。

## 回答<a name="section5223998"></a>

根据返回消息进行判断原因：

-   检查账号是否已通过实名认证。若没有，请先实名认证。
-   检查账户是否已欠费或余额不足。若欠费，请先续费。
-   检查账号是否拥有权限，若无权限，请授予对应的操作权限。
-   若当前用户所创建的桶已达到上限100个，删除一些闲置的桶再创建。
-   若是当前桶名已存在，则更换桶名再创建。
